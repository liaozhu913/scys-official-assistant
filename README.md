# 生财有术官网助手

面向生财有术官网的油猴脚本，主功能是图片增强：点击正文图片后可放大查看、自由缩放、拖拽平移，并支持上一张 / 下一张切换。

## 功能

- 图片点击放大查看
- 图片自由缩放、拖拽、适应窗口
- 同一文章内图片上一张 / 下一张切换
- 高级功能设置面板
- 授权后可显示 / 隐藏 Markdown 浮窗按钮
- 私用 Markdown 功能：查看 MD、复制 MD、下载 MD

## 安装

1. 安装 Tampermonkey / Violentmonkey 等用户脚本管理器。
2. 打开 `生财有术官网助手-1.0.user.js`。
3. 在用户脚本管理器中确认安装。

## 使用

- 在 `https://scys.com/*` 页面中，点击正文图片即可打开图片查看器。
- 在脚本管理器菜单中点击 `高级功能`，可复制随机设备码、输入解锁密钥，并设置是否显示 Markdown 浮窗按钮。
